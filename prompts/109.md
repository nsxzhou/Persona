# 109. 失忆/身份重构剧情线

## 提示词内容

```json
{
  "task": "outline_amnesia_arc",
  "mc_status": "Lost memory, woke up in enemy territory",
  "clues": [
    "A mysterious key in pocket",
    "Muscle memory of advanced combat skills",
    "Recurring nightmare of a burning city"
  ],
  "conflict": "New friends vs. Old identity (which might be evil)",
  "reveal": "MC was the top assassin sent to kill the current leader"
}
```

## 使用场景
悬疑/特工文。通过寻找记忆揭开惊天阴谋。

## 最佳实践要点
1.  **本能反应**：通过身体本能（如瞬间制服敌人）暗示主角的不凡过去。
2.  **身份冲突**：新建立的羁绊与旧身份的任务产生冲突，制造戏剧张力。

## 示例输入
无。
