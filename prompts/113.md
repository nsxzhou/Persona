# 113. 五感环境氛围扩写

## 提示词内容

```json
{
  "task": "expand_sensory_details",
  "setting": "Rainy Alleyway / Busy Market / Alien Jungle",
  "senses": {
    "Sight": "Neon reflections in puddles",
    "Sound": "Distant sirens, rain hitting tin roofs",
    "Smell": "Ozone, rotting garbage, street food",
    "Touch": "Cold damp air, sticky humidity",
    "Taste": "Metallic taste of blood / Dust in mouth"
  },
  "instruction": "Combine these into an immersive paragraph."
}
```

## 使用场景
场景描写。增强读者的沉浸感。

## 最佳实践要点
1.  **多维打击**：不要只写看到的，必须包含听觉、嗅觉等容易被忽略的感官。
2.  **情感投射**：环境描写应反映人物的心境（如雨天衬托悲伤）。

## 示例输入
无。
