# 118. 地下城美食食谱 (迷宫饭风)

## 提示词内容

```json
{
  "task": "create_monster_recipe",
  "monster": "Giant Scorpion / Slime / Mandrake",
  "preparation": "Remove poison sac / Dry in sun / Boil with holy water",
  "cooking_method": "Stir-fry with garlic / Stew",
  "taste_profile": "Tastes like lobster but spicy / Jelly-like texture",
  "buff_effect": "Increases poison resistance for 1 hour"
}
```

## 使用场景
美食/冒险文。设计新奇有趣的怪物料理。

## 最佳实践要点
1.  **去毒处理**：详细描写处理怪物的过程，增加真实感和猎奇感。
2.  **口感类比**：用现实食物（如龙虾、果冻）类比口感，方便读者想象。

## 示例输入
无。
