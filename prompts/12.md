# 12. 黄金三章细纲

## 提示词内容

```json
{
  "task": "outline_golden_three",
  "core_hook": "{{hook}}",
  "output_structure": {
    "chapter_1": {
      "title": "Impactful Title",
      "scenes": ["Scene 1: Status Quo (Misery)", "Scene 2: Inciting Incident", "Scene 3: Gold Finger Activation"],
      "hook": "End of chapter cliffhanger"
    },
    "chapter_2": {
      "title": "Impactful Title",
      "scenes": ["Scene 1: Understanding Cheat", "Scene 2: First Small Slap/Success", "Scene 3: Escalation"],
      "hook": "End of chapter cliffhanger"
    },
    "chapter_3": {
      "title": "Impactful Title",
      "scenes": ["Scene 1: Conflict Climax", "Scene 2: Reward/Loot", "Scene 3: New Goal Set"],
      "hook": "End of chapter cliffhanger"
    }
  }
}
```

## 使用场景
开篇创作。设计决定完读率的前三章。

## 最佳实践要点
1.  **钩子原则 (Hook Principle)**：每章结尾必须有 `hook`（悬念），强迫读者翻页。
2.  **紧凑结构**：前三章完成“压抑-金手指-小高潮”的完整闭环。

## 示例输入
将 `{{hook}}` 替换为“重生回高考前一天，获得透视眼”。
