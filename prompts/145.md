# 145. 无限流主神任务发布机制

## 提示词内容

```json
{
  "task": "generate_god_mission",
  "world": "Zombie Apocalypse / Jurassic Park",
  "main_quest": "Survive for 7 days / Retrieve the T-Rex egg",
  "side_quest": "Kill 100 zombies (Optional)",
  "penalty": "Obliteration / Deduct 1000 points",
  "twist": "The T-Rex is actually a robot / Zombies can run"
}
```

## 使用场景
无限流。设计清晰且充满陷阱的任务系统。

## 最佳实践要点
1.  **文字游戏**：主神发布的任务往往有歧义，利用这一点坑害轮回者。
2.  **难度分级**：区分主线（生存）和支线（贪婪），考验主角的决策。

## 示例输入
无。
