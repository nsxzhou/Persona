# 15. 首秀数据模拟

## 提示词内容

```json
{
  "task": "simulate_launch_metrics",
  "inputs": {
    "title": "{{title}}",
    "intro": "{{intro}}",
    "first_3_chapters_summary": "{{summary}}"
  },
  "simulation_output": {
    "predicted_ctr": "Estimated Click-Through Rate (%)",
    "predicted_retention": "Estimated 100k Word Retention (%)",
    "strengths": ["List of strong points"],
    "weaknesses": ["List of weak points (e.g., slow start, confusing intro)"],
    "actionable_advice": "Specific changes to improve metrics"
  }
}
```

## 使用场景
发书前自测。预测作品的市场表现。

## 最佳实践要点
1.  **反馈循环**：利用 AI 的评估能力作为“模拟读者”，提前发现问题。
2.  **指标化**：关注 CTR 和留存率这两个核心指标，使优化方向更明确。

## 示例输入
填入书名、简介和前三章梗概。
