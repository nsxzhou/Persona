# 151. 意识流/心理独白写法

## 提示词内容

```json
{
  "task": "write_stream_of_consciousness",
  "character": "A soldier with PTSD / A genius solving a puzzle",
  "trigger": "A loud bang / A complex formula",
  "style": "Fragmented sentences, sensory overload, jumping topics",
  "content": "Flashbacks mixed with present reality, rapid associations."
}
```

## 使用场景
深度人物刻画。描写角色混乱或高速运转的思维。

## 最佳实践要点
1.  **碎片化**：打破语法规范，用短语、词块模拟思维的跳跃。
2.  **感官混杂**：将记忆的碎片与当下的感官体验（声音、气味）交织。

无。
