# 153. 魔法/阵法符文视觉设计

## 提示词内容

```json
{
  "task": "visualize_magic_rune",
  "spell": "Fireball / Teleportation",
  "shape": "Circle within a triangle / Spirals",
  "color": "Burning orange / Void purple",
  "movement": "Rotating rings, pulsating light",
  "sound": "Humming, cracking",
  "instruction": "Describe the casting process visually."
}
```

## 使用场景
奇幻/玄幻文。让施法过程具有画面感。

## 最佳实践要点
1.  **几何美学**：用几何图形（圆、三角、螺旋）构建魔法阵的结构。
2.  **动态过程**：描写符文的点亮、旋转、破碎等动态变化。

## 示例输入
无。
