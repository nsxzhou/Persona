# 159. 史诗感演说/战前动员

## 提示词内容

```json
{
  "task": "write_epic_speech",
  "speaker": "King / General / Rebel Leader",
  "audience": "Soldiers facing death",
  "theme": "Freedom / Honor / Survival",
  "structure": [
    "Acknowledge the fear",
    "Remind them what they fight for (Family/Home)",
    "Promise glory or victory",
    "Call to action (Roar)"
  ],
  "tone": "Inspiring, gritty, loud"
}
```

## 使用场景
战争/争霸文。提升士气，制造高潮。

## 最佳实践要点
1.  **共情**：先承认恐惧，再升华勇气，建立与听众的情感连接。
2.  **短促有力**：结尾的口号必须简短有力，适合万人齐呼。

无。
