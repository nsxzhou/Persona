# 160. 平行宇宙差异点设计

## 提示词内容

```json
{
  "task": "design_parallel_universe",
  "divergence_point": "Germany won WWII / Dinosaurs didn't go extinct / Magic exists",
  "current_state": "Dystopian police state / Tribal lizard society / Magitech city",
  "mc_reaction": "Confusion due to subtle differences (e.g., currency, flag)",
  "twist": "MC's counterpart in this world is the villain"
}
```

## 使用场景
科幻/穿越文。构建平行世界的差异。

## 最佳实践要点
1.  **蝴蝶效应**：一个历史节点的改变（分歧点）应导致现在的巨大差异。
2.  **自我对立**：主角与平行世界的自己对立，是经典的戏剧冲突来源。

## 示例输入
无。
