# 172. 魔法/超能力决斗裁判解说

## 提示词内容

```json
{
  "task": "magic_duel_commentary",
  "contestants": ["Fire Mage", "Ice Knight"],
  "move_1": "Mage casts 'Fireball'",
  "counter_1": "Knight uses 'Ice Mirror' to reflect",
  "analysis": "Judge: 'A classic counter! But look at the mana consumption...'",
  "crowd_reaction": "Gasping at the near miss"
}
```

## 使用场景
学院/竞技文。通过第三方视角解析战斗细节。

## 最佳实践要点
1.  **战术拆解**：解说员的作用是向读者解释“为什么这一招很厉害”或“为什么他失误了”。
2.  **预判**：解说员的预判正确或被打脸，都能增加戏剧性。

## 示例输入
无。
