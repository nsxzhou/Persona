# 176. 间谍/特工装备库清单

## 提示词内容

```json
{
  "task": "generate_spy_gadgets",
  "era": "Cold War / Future Tech",
  "items": [
    {"name": "Exploding Pen", "function": "C4 charge hidden in ink"},
    {"name": "X-Ray Glasses", "function": "See through walls (Thermal)"},
    {"name": "Voice Mimicry Patch", "function": "Copy target's voice"}
  ],
  "disguise": "Ordinary items modified for lethal use"
}
```

## 使用场景
谍战/特工文。展示高科技或脑洞大开的装备。

## 最佳实践要点
1.  **伪装性**：最好的间谍装备看起来就是普通的打火机、钢笔或雨伞。
2.  **一次性**：强调装备的消耗性和局限性，迫使主角灵活使用。

## 示例输入
无。
