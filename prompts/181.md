# 181. 傲娇 (Tsundere) 台词生成器

## 提示词内容

```json
{
  "task": "generate_tsundere_lines",
  "context": "Giving a gift / Helping MC",
  "intensity": "Mild (Blushing) / Heavy (Violent)",
  "lines": [
    "It's not like I did this for you or anything!",
    "Don't get the wrong idea, I just had extra.",
    "Baka! (Idiot!)"
  ],
  "subtext": "She actually loves him very much."
}
```

## 使用场景
二次元/轻小说。生成标准的傲娇角色互动。

## 最佳实践要点
1.  **口是心非**：台词必须是否定好意的（“才不是为了你”），但行为是诚实的。
2.  **掩饰害羞**：用愤怒或暴躁来掩饰害羞，这是傲娇的核心萌点。

## 示例输入
无。
