# 19. 年代感细节注入 (70/80年代)

## 提示词内容

```json
{
  "task": "inject_period_details",
  "era": "1980s China",
  "setting": "Rural Village / State-owned Factory",
  "categories": {
    "items": ["Enamel mug", "Sewing machine", "Green army coat"],
    "food": ["Cornbread", "Braised pork (Rare treat)", "Malted milk"],
    "slang": ["Iron rice bowl", "Speculator"],
    "social_event": "Open-air movie, College entrance exam"
  },
  "instruction": "Weave these elements naturally into a paragraph describing the protagonist's morning routine."
}
```

## 使用场景
年代文/种田文创作。增强故事的历史质感和真实度。

## 最佳实践要点
1.  **细节沉浸**：通过具体的物品、食物和俚语，而非宏大的历史叙事，来体现年代感。
2.  **场景融合**：要求 AI 将元素自然融入日常场景，避免堆砌词汇。

## 示例输入
直接使用或修改 `era` 和 `setting`。
