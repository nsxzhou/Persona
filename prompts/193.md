# 193. 杀手/刺客组织的信条与戒律

## 提示词内容

```json
{
  "task": "create_assassin_creed",
  "organization": "The Faceless Men",
  "tenets": [
    "Nothing is true, everything is permitted.",
    "Hide in plain sight.",
    "Never harm an innocent."
  ],
  "initiation": "Cutting off a finger / Erasing past identity",
  "sanctuary": "A continental hotel where violence is forbidden"
}
```

## 使用场景
动作/都市异能。构建有逼格的地下组织。

## 最佳实践要点
1.  **仪式感**：通过特殊的入会仪式（断指、纹身）强化归属感。
2.  **中立区**：设立绝对安全的避难所（如和平饭店），是各路杀手互动的舞台。

## 示例输入
无。
