# 195. 龙与地下城 (DND) 怪物遭遇战设计

## 提示词内容

```json
{
  "task": "design_dnd_encounter",
  "party_level": "Level 5",
  "monster": "Owlbear (Challenge Rating 3) + 2 Goblins",
  "terrain": "Dense forest, difficult terrain",
  "tactics": "Owlbear rushes the tank, Goblins shoot from trees",
  "loot": "Owlbear beak, Goblin rusty daggers"
}
```

## 使用场景
西幻/游戏文。设计平衡且有趣的战斗遭遇。

## 最佳实践要点
1.  **地形互动**：怪物会利用地形（如地精躲树上），增加战斗策略性。
2.  **组合拳**：单一怪物容易被围殴，通过主怪+小怪的组合增加难度。

无。
