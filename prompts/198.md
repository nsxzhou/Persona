# 198. 灵异照片/视频描述生成

## 提示词内容

```json
{
  "task": "describe_ghost_photo",
  "subject": "Group selfie at a party",
  "anomaly": "An extra hand on a shoulder / A face in the dark window",
  "subtlety": "Hard to see at first glance, terrifying once noticed",
  "reaction": "Characters arguing if it's a glitch or a ghost"
}
```

## 使用场景
灵异/恐怖文。通过视觉媒介传递恐怖感。

## 最佳实践要点
1.  **细思极恐**：恐怖元素不要太明显，要藏在角落或背景中。
2.  **逐渐发现**：描写角色放大照片、调整亮度后发现真相的过程。

## 示例输入
无。
