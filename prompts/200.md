# 200. 全球进化/游戏化开篇引导语

## 提示词内容

```json
{
  "task": "write_system_announcement",
  "voice": "Cold, mechanical, god-like",
  "message": [
    "Welcome to Earth 2.0.",
    "Survival is your only objective.",
    "Loading resources... 10%... 100%.",
    "Good luck, players."
  ],
  "effect": "Appears directly in everyone's retina / Sky turns into a screen"
}
```

## 使用场景
末世/游戏化/无限流。经典的开篇“天音”。

## 最佳实践要点
1.  **不可抗力**：强调声音直接出现在脑海或视网膜，无法屏蔽，确立系统的绝对权威。
2.  **简洁冷酷**：宣告必须简短，不解释原因，直接宣布规则。

## 示例输入
无。
