# 205. 麦格芬 (McGuffin) 设计

## 提示词内容

```json
{
  "task": "design_mcguffin",
  "name": "The Chrono-Crystal / The Lost Treaty",
  "appearance": "A pulsing blue shard / A simple piece of paper",
  "function": "Unknown power / Grants legitimacy to the throne",
  "location": "Hidden in plain sight / Guarded by a dragon",
  "conflict": "Everyone wants it, but no one knows exactly how to use it"
}
```

## 使用场景
冒险/夺宝文。设计推动剧情的核心道具。

## 最佳实践要点
1.  **功能模糊**：麦格芬的具体功能往往不重要，重要的是“每个人都想要它”。
2.  **象征意义**：它代表了权力、永生或最终的胜利。

## 示例输入
无。
