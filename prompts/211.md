# 211. 机甲/外骨骼自定义方案

## 提示词内容

```json
{
  "task": "customize_mecha",
  "base_frame": "Light Recon / Heavy Assault",
  "weapons": ["Shoulder-mounted Railgun", "Plasma Saber", "Missile Pods"],
  "paint_job": "Matte Black with Red Stripes (Char Aznable style)",
  "special_system": "Overdrive Mode (Increases speed by 300%, risks overheating)",
  "name": "The Crimson Comet"
}
```

## 使用场景
科幻/机甲文。展现主角对装备的独特理解和改造。

## 最佳实践要点
1.  **权衡**：改装不是简单的堆砌，而是取舍（如牺牲装甲换取速度）。
2.  **涂装**：个性化的涂装和命名赋予机甲灵魂。

## 示例输入
无。
