# 225. 赛博精神病 (Cyberpsychosis) 发作描写

## 提示词内容

```json
{
  "task": "describe_cyberpsychosis",
  "symptoms": "Hallucinations / Loss of empathy / Sensory overload",
  "trigger": "Installing too many mods / Emotional trauma",
  "vision": "Seeing people as demons or targets",
  "action": "Laughing while destroying everything",
  "tragedy": "Killing loved ones without realizing it"
}
```

## 使用场景
赛博朋克。描写过度改造带来的精神崩溃。

## 最佳实践要点
1.  **异化**：描写主角逐渐觉得自己不再是人，而是机器或神。
2.  **悲剧性**：赛博精神病往往伴随着对自己珍视之物的毁灭。

## 示例输入
无。
