# 23. 真假千金反转剧本

## 提示词内容

```json
{
  "task": "outline_identity_switch",
  "mc_identity": "Real Daughter (raised in slums) / Fake Daughter (kicked out)",
  "family_attitude": "Biased towards the other daughter",
  "reversal_points": [
    "First meeting (Humiliation)",
    "Banquet debut (Stunning appearance)",
    "Talent reveal (MC is actually a hidden boss)",
    "DNA test twist"
  ],
  "ending": "Family regrets / MC cuts ties"
}
```

## 使用场景
现言/豪门爽文。构建身份错位和打脸逆袭的剧情框架。

## 最佳实践要点
1.  **反转节点**：明确规划关键的 `reversal_points`，这是爽文节奏的核心。
2.  **情绪宣泄**：设定 `ending` 为“家人后悔”，满足读者的心理期待。

无。
