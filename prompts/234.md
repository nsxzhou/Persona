# 234. 虚拟网游：公会战指挥艺术

## 提示词内容

```json
{
  "task": "write_guild_war_command",
  "objective": "Capture the Flag / Siege the Castle",
  "tactic": "Zerg rush / Pincer movement / Stealth backdoor",
  "comms": [
    "Group 1, hold the bridge!",
    "Rogues, flank left!",
    "Healers, pop ults NOW!"
  ],
  "chaos": "Spell effects blinding the screen, lag, shouting"
}
```

## 使用场景
网游/电竞文。描写大规模团战的指挥。

## 最佳实践要点
1.  **大局观**：指挥官看到的不是一个人，而是资源交换和位置控制。
2.  **执行力**：描写队员对指令的瞬间反应（或失误）。

## 示例输入
无。
