# 24. 读心术/吃瓜系统群像描写

## 提示词内容

```json
{
  "task": "write_mind_reading_scene",
  "mc_action": "Thinking/System Broadcast",
  "bystanders": ["Emperor", "Concubines", "Ministers"],
  "secret_revealed": "{{secret}}",
  "reactions": {
    "surface": "Maintaining poker face",
    "inner": "Panic / Shock / Holding back laughter"
  },
  "outcome": "Misunderstanding resolved or villain exposed via inner voice"
}
```

## 使用场景
沙雕/吃瓜流爽文。描写心声泄露引发的群像反应。

## 最佳实践要点
1.  **反差萌**：重点描写 `surface` 和 `inner` 的反差，制造喜剧效果。
2.  **信息不对称**：利用心声泄露打破信息壁垒，推动剧情发展。

## 示例输入
将 `{{secret}}` 替换为“皇帝其实怕猫”。
