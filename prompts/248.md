# 248. 魔法体系平衡：代价与限制

## 提示词内容

```json
{
  "task": "balance_magic_system",
  "magic_type": "Blood Magic / Time Magic",
  "cost": "Lifespan / Memory / Sanity",
  "limit": "Can only be used at night / Requires a catalyst",
  "loophole": "MC uses enemy's blood / MC has perfect memory recall",
  "consequence": "Overuse leads to turning into a monster"
}
```

## 使用场景
奇幻/系统文。设计严谨且有深度的魔法规则。

## 最佳实践要点
1.  **等价交换**：越强大的魔法，代价越昂贵，防止战力崩坏。
2.  **卡BUG**：主角的金手指通常表现为能够规避或减轻这种代价。

## 示例输入
无。
