# 261. 规则怪谈：隐晦剧情 (纸条/日记)

## 提示词内容

```json
{
  "task": "write_narrative_via_notes",
  "items": [
    {
      "type": "Torn Diary Page",
      "content": "Don't trust the red uniform... I saw him eating...",
      "location": "Under the bed"
    },
    {
      "type": "Sticky Note",
      "content": "Rule 3 is a LIE. Run.",
      "location": "Behind the mirror"
    },
    {
      "type": "Blood Message",
      "content": "It's still here.",
      "location": "Ceiling"
    }
  ],
  "style": "Fragmented, desperate, contradictory",
  "goal": "Reveal the hidden truth about the location without direct exposition"
}
```

## 使用场景
规则怪谈/无限流/悬疑。通过碎片化信息拼凑真相。

## 最佳实践要点
1.  **不可靠叙述**：纸条的主人可能已经疯了，或者是在误导。
2.  **细思极恐**：通过留白让读者自己脑补恐怖的真相。

## 示例输入
场景：废弃的精神病院。
