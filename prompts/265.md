# 265. 悬疑：多重人格会议 (Inner Meeting)

## 提示词内容

```json
{
  "task": "write_inner_personality_meeting",
  "location": "Mind Palace / Round Table / Dark Room",
  "personalities": [
    {"name": "The Protector", "trait": "Violent, strong"},
    {"name": "The Child", "trait": "Scared, holds memories"},
    {"name": "The Strategist", "trait": "Cold, logical"},
    {"name": "The Host", "trait": "Confused, unaware"}
  ],
  "topic": "Who takes control next? / How to deal with the threat?",
  "conflict": "Protector wants to kill, Strategist wants to negotiate"
}
```

## 使用场景
悬疑/心理/无限流。具象化多重人格的内部交流。

## 最佳实践要点
1.  **性格鲜明**：每个人格的语气、立场必须截然不同。
2.  **主导权争夺**：会议的核心往往是谁来掌控身体。

## 示例输入
主角面临生死危机，内部人格召开紧急会议。
