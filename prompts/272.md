# 272. 反派：最终BOSS的临终遗言

## 提示词内容

```json
{
  "task": "write_villain_last_words",
  "tone": "Regretful / Defiant / Mocking / Enlightened",
  "revelation": "I was holding back a greater evil / The world is fake / You are just like me",
  "action": "Hands over a key item / Dies standing up / Disintegrates",
  "impact": "MC questions their own justice / The victory feels hollow"
}
```

## 使用场景
玄幻/奇幻/大结局。升华主题，反派的高光时刻。

## 最佳实践要点
1.  **格局打开**：反派的死往往揭示了更大的世界观或阴谋。
2.  **不落俗套**：避免“我还会回来的”这种低级台词。

## 示例输入
魔王被击败后，笑着告诉勇者：我是为了封印地下的古神才统治世界的。
