# 273. 系统：任务失败惩罚机制

## 提示词内容

```json
{
  "task": "write_system_punishment",
  "mission": "Confess to the school beauty",
  "failure_condition": "Rejection / Time out",
  "punishment_type": "Embarrassment / Physical change / Loss of wealth",
  "examples": [
    "Bark like a dog in public for 1 minute",
    "Lose 5cm of height",
    "All cash in wallet turns to leaves",
    "Forced to wear a skirt for a day"
  ],
  "system_tone": "Gloating / Indifferent / Sadistic"
}
```

## 使用场景
系统流/搞笑/都市。通过惩罚机制推动主角行动，制造笑料。

## 最佳实践要点
1.  **社死为主**：现在的网文倾向于“社会性死亡”而非肉体抹杀。
2.  **紧迫感**：惩罚越变态，主角完成任务的动力越足。

## 示例输入
系统发布任务：在一小时内花光一百万，失败则当街跳极乐净土。
