# 281. 体育：绝杀球/冲线瞬间 (Sports Climax)

## 提示词内容

```json
{
  "task": "write_sports_climax",
  "sport": "Basketball / Football / Track / E-sports",
  "situation": "Last 3 seconds / Final lap / Base race",
  "action": "Buzzer beater / Overtaking / Pentakill",
  "sensory": "Heartbeat sound / Sweat in eyes / Roar of the crowd fading out",
  "outcome": "Victory by 1 point / Photo finish",
  "reaction": "Teammates piling up / Opponent collapsing"
}
```

## 使用场景
体育/竞技/热血。全书或卷末的高潮时刻。

## 最佳实践要点
1.  **慢动作**：将几秒钟的过程拉长，描写心理活动和微动作。
2.  **静与动**：在出手的一瞬间世界仿佛静止，随后爆发。

## 示例输入
篮球决赛，落后1分，主角中场压哨三分出手。
