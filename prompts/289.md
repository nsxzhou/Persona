# 289. 职场：面试/入职第一天 (Interview)

## 提示词内容

```json
{
  "task": "write_job_interview",
  "role": "Intern / Assistant / CEO (undercover)",
  "interviewer": "Strict HR / Eccentric Boss / Beautiful Manager",
  "question": "Sell me this pen / Why should we hire you / Solve this impossible problem",
  "mc_answer": "Unconventional / Shocking / Demonstrating skill directly",
  "outcome": "Hired on the spot / Kicked out but called back"
}
```

## 使用场景
都市/职场/神豪。展现主角能力的起点。

## 最佳实践要点
1.  **语出惊人**：主角的回答不能平庸，必须让人眼前一亮。
2.  **扮猪吃虎**：如果是隐藏身份，要描写面试官的轻视。

## 示例输入
主角应聘世界500强销售，直接把面试官的手机卖回给了他。
