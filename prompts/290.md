# 290. 恐怖：电梯惊魂 (Elevator Horror)

## 提示词内容

```json
{
  "task": "write_elevator_horror",
  "situation": "Stuck between floors / Going to the wrong floor (B4 / 18th)",
  "anomaly": "Passenger count mismatch / Mirror reflection wrong / Infinite corridor outside",
  "sensory": "Flickering lights / Heavy breathing / Temperature drop",
  "passenger": "A woman with back turned / A child staring at corner",
  "climax": "The doors open to darkness / The passenger whispers something"
}
```

## 使用场景
灵异/悬疑/无限流。密闭空间恐惧症的经典应用。

## 最佳实践要点
1.  **数字心理**：利用楼层数字（4、18、-1）的暗示。
2.  **无法逃离**：电梯是移动的棺材，强调这种无助感。

## 示例输入
深夜独自坐电梯，电梯明明在上升，显示屏却在倒数。
