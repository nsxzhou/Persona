# 298. 灵异：鬼打墙/循环空间 (Loop)

## 提示词内容

```json
{
  "task": "write_loop_horror",
  "setting": "Staircase / Forest path / Highway tunnel",
  "sign": "Seeing the same mark again / Time not moving",
  "escalation": "Running faster but staying in place / Fog getting thicker",
  "panic": "Breathing heavily / Screaming / Loss of direction",
  "breakthrough": "Realizing the rule / Closing eyes / Walking backwards"
}
```

## 使用场景
灵异/无限流/惊悚。制造绝望感和困兽之斗的氛围。

## 最佳实践要点
1.  **参照物**：通过一个参照物（如丢在地上的烟头）确认循环。
2.  **心理崩溃**：描写主角从疑惑到恐慌再到绝望的过程。

## 示例输入
主角在楼梯间向下跑，却总是回到四楼。
