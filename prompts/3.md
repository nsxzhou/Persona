# 3. 赛道热度分析 (2024-2026)

## 提示词内容

```json
{
  "task": "analyze_market_trends",
  "platform": "Tomato Novel",
  "year_range": "2024-2026",
  "focus_areas": [
    "Male Lead (Urban, Sci-Fi, Suspense)",
    "Female Lead (Romance, Modern, Historical)"
  ],
  "output_requirements": {
    "top_3_genres": "Most popular genres with high traffic",
    "rising_stars": "Niche genres showing rapid growth (e.g., Rule Weird + Infinite Flow)",
    "saturated_tropes": "Overused tropes to avoid or twist",
    "audience_preferences": "What readers are currently craving (e.g., Logical consistency, Face-slapping)"
  }
}
```

## 使用场景
在选题策划阶段使用。帮助作者了解当前市场风向，避开红海题材，寻找蓝海或微创新点。

## 最佳实践要点
1.  **思维链 (Chain of Thought)**：引导 AI 分析“热门”背后的原因（用户偏好）。
2.  **数据驱动模拟**：虽然 AI 不能实时联网（除非使用联网搜索模式），但可以通过训练数据模拟市场趋势分析。
3.  **明确输出目标**：指定输出 Top 3、黑马题材和饱和套路，使结果更具可操作性。

## 示例输入
直接复制上述 JSON 代码块发送给 DeepSeek。如需特定年份或平台，可修改 `year_range` 或 `platform`。
