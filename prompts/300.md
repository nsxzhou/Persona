# 300. 完结：全书完结感言 (Final Author Note)

## 提示词内容

```json
{
  "task": "write_author_final_note",
  "tone": "Grateful / Emotional / Relieved / Professional",
  "content": [
    "Thanking readers",
    "Explaining the ending (optional)",
    "Sharing writing hardships",
    "Teasing next book",
    "Asking for support (subscriptions/votes)"
  ],
  "style": "Sincere / Humorous / Philosophical",
  "length": "Short and sweet / Long essay"
}
```

## 使用场景
全书完结。作者与读者的最后一次正式交流。

## 最佳实践要点
1.  **真诚**：真诚最能打动人，感谢读者的陪伴。
2.  **引流**：巧妙地预告新书，留住粉丝。

## 示例输入
写一篇完结感言，感谢读者三年的陪伴，预告下本是仙侠题材。
