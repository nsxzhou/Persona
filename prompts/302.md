# 302. 恋爱模拟：好感度事件 (Dating Sim Event)

## 提示词内容

```json
{
  "task": "write_dating_sim_event",
  "character": "The Student Council President (Tsundere)",
  "location": "Rooftop / Library / Festival",
  "affection_level": "Level 3 (Acquaintance -> Friend)",
  "trigger": "MC finds her feeding a stray cat",
  "choices": [
    {"option": "Tease her", "result": "+Affection, -Mood"},
    {"option": "Help her", "result": "++Affection, CG unlocked"},
    {"option": "Ignore her", "result": "No change"}
  ],
  "dialogue": "Blushing and stammering"
}
```

## 使用场景
恋爱/综漫/系统流。模拟 Galgame 的分支剧情和好感度系统。

## 最佳实践要点
1.  **选项设计**：选项要对应不同的性格反馈。
2.  **CG时刻**：描写关键的“心动瞬间”画面。

## 示例输入
攻略对象：高冷学姐。场景：雨天图书馆。
