# 303. 克苏鲁：调查员的疯狂日志 (Sanity Loss)

## 提示词内容

```json
{
  "task": "write_investigator_journal",
  "day": "Day 1 -> Day 7 -> Day ???",
  "symptoms": "Insomnia / Hallucinations / Obsessive writing",
  "discovery": "The geometry of the room is wrong / The rats speak in Latin",
  "style": "Handwriting gets messier / Sentences cut off",
  "climax": "The door is opening... I see it...",
  "last_entry": "Ph'nglui mglw'nafh..."
}
```

## 使用场景
克苏鲁/悬疑/惊悚。通过第一人称日志展现理智的崩溃。

## 最佳实践要点
1.  **渐进式恐惧**：从正常的记录逐渐变成疯狂的呓语。
2.  **不可靠叙述**：记录者看到的“真相”可能只是幻觉。

## 示例输入
一名考古学家在发掘古墓期间的七日日记。
