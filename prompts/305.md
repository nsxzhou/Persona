# 305. 美食探店：街头小吃描写 (Food Review)

## 提示词内容

```json
{
  "task": "write_street_food_review",
  "food": "Stinky Tofu / Takoyaki / Spicy Crayfish",
  "sensory": {
    "sight": "Golden crispy skin / Oil dripping",
    "smell": "Pungent but appetizing / Spicy aroma",
    "sound": "Crunching / Sizzling",
    "taste": "Explosion of flavor / Numbing spice"
  },
  "atmosphere": "Noisy night market / Plastic stools / Long queue",
  "reaction": "Sweating but can't stop eating"
}
```

## 使用场景
美食/都市/直播。用文字勾起读者的食欲。

## 最佳实践要点
1.  **色香味俱全**：调动所有感官，特别是听觉（酥脆声）。
2.  **市井气**：环境描写要接地气，嘈杂但热闹。

## 示例输入
主角在深夜大排档吃一份爆辣小龙虾。
