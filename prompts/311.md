# 311. 奇幻：魔法卷轴/药剂说明书 (Item Description)

## 提示词内容

```json
{
  "task": "write_item_description",
  "item": "Potion of Invisibility / Scroll of Fireball",
  "appearance": "Bubbling purple liquid / Ancient parchment with glowing runes",
  "effect": "Turns user invisible for 10 minutes",
  "side_effect": "User smells like rotten eggs / Hair falls out",
  "flavor_text": "'Best used when running away from angry dragons.' - Archmage X",
  "rarity": "Common / Rare / Legendary"
}
```

## 使用场景
西幻/游戏/系统。增加物品的趣味性和世界观深度。

## 最佳实践要点
1.  **风味文本**：像《魔兽世界》或《暗黑破坏神》那样的趣味道具描述。
2.  **副作用**：强大的道具往往伴随着坑爹的副作用。

## 示例输入
一瓶喝了能变强壮但会变秃的药剂。
