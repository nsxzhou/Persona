# 323. 都市：直播带货 (Live Streaming Sales)

## 提示词内容

```json
{
  "task": "write_livestream_sales",
  "product": "Magic Pill / Futuristic Tech / Weird Snack",
  "host_style": "High energy / Shouting / Storytelling",
  "audience": "Skeptical -> Excited -> Frenzy",
  "sales_tactic": "Limited time offer / Fake discount / Emotional manipulation",
  "climax": "Sold out in 3 seconds / Server crash"
}
```

## 使用场景
都市/系统/神豪。结合时事的带货情节。

## 最佳实践要点
1.  **煽动性**：主播的话术要有极强的感染力（“家人们！”）。
2.  **疯抢**：描写弹幕刷屏和库存秒空的场面。

## 示例输入
主角在修仙界直播带货“洗髓丹”，全网疯抢。
