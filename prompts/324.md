# 324. 悬疑：梦境解析 (Dream Analysis)

## 提示词内容

```json
{
  "task": "write_dream_analysis",
  "dream_content": "Falling from a tower / Teeth falling out / Being chased by a shadow",
  "interpretation": "Fear of loss / Anxiety / Guilt",
  "clue": "The shadow in the dream has the same tattoo as the killer",
  "method": "Freudian / Jungian / Supernatural",
  "revelation": "The dream is a suppressed memory"
}
```

## 使用场景
悬疑/心理/灵异。通过梦境挖掘潜意识线索。

## 最佳实践要点
1.  **象征意义**：梦中的元素往往象征着现实中的人和事。
2.  **恐怖谷**：梦境的逻辑往往荒诞且令人不安。

## 示例输入
心理医生分析主角的噩梦，发现他目击了凶案现场。
