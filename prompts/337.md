# 337. 游戏：公会内讧 (Guild Drama)

## 提示词内容

```json
{
  "task": "write_guild_drama",
  "cause": "Loot distribution / Spy / Leadership struggle / Jealousy",
  "factions": "The Founder vs The Rich Sponsor / Veterans vs Newbies",
  "event": "Guild chat argument / Mass quitting / Griefing",
  "mc_role": "Mediator / Instigator / Observer",
  "outcome": "Guild disbanding / New guild formed / Purge"
}
```

## 使用场景
网游/电竞/职场。虚拟世界的办公室政治。

## 最佳实践要点
1.  **利益纠葛**：一切矛盾的根源都是利益（装备、地位）。
2.  **人心**：展现人性的贪婪和背叛。

## 示例输入
第一公会因为一件神器分配不均，爆发内战解散。
