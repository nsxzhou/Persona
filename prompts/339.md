# 339. 职场：背锅与甩锅 (Scapegoating)

## 提示词内容

```json
{
  "task": "write_office_scapegoating",
  "incident": "Project failure / Data leak / Lost client",
  "villain": "Incompetent Manager / Cunning Colleague",
  "tactic": "Blaming the intern (MC) / Falsifying emails",
  "confrontation": "Meeting room argument",
  "mc_counter": "Bringing out the recorded audio / Backed up logs / CEO's support",
  "outcome": "Villain fired / MC promoted"

```

## 使用场景
都市/职场/爽文。职场生存指南。

## 最佳实践要点
1.  **证据**：职场反击的核心是证据链。
2.  **隐忍**：前期忍辱负重，收集证据，最后绝地反击。

## 示例输入
上司把项目失败的责任推给主角，主角当场拿出聊天记录打脸。
