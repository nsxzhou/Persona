# 340. 写作工具：情感曲线设计 (Emotional Arc)

## 提示词内容

```json
{
  "task": "design_emotional_arc",
  "character": "MC / Heroine",
  "start_state": "Hopeful / Naive / Broken",
  "inciting_incident": "Betrayal / Tragedy / Success",
  "low_point": "All is lost / Dark Night of the Soul",
  "climax": "Realization / Sacrifice / Triumph",
  "end_state": "Mature / Cynical / Healed",
  "pacing": "Chapter 1 to Chapter 50"
}
```

## 使用场景
通用/大纲/人设。规划角色情感变化的轨迹。

## 最佳实践要点
1.  **起伏**：情感不能是一条直线，要有波峰波谷。
2.  **成长**：经历事件后，角色的性格必须发生改变。

## 示例输入
设计一个“天真少年”变成“冷血复仇者”的情感曲线。
