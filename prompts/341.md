# 341. 细纲：单章情绪心电图 (Emotional ECG)

## 提示词内容

```json
{
  "task": "create_chapter_emotional_beat",
  "chapter_goal": "MC realizes he has been betrayed",
  "beats": [
    {"type": "Opening", "emotion": "Calm/Happy", "action": "Celebrating victory"},
    {"type": "Inciting", "emotion": "Confusion", "action": "Noticing a small detail wrong"},
    {"type": "Rising", "emotion": "Suspicion/Fear", "action": "Investigating the detail"},
    {"type": "Climax", "emotion": "Shock/Anger", "action": "Confronting the traitor"},
    {"type": "Ending", "emotion": "Despair/Determination", "action": "Traitor escapes, MC is left alone"}
  ],
  "pacing": "Slow build-up -> Fast climax"
}
```

## 使用场景
细纲/写作。规划单章的情绪起伏，避免流水账。

## 最佳实践要点
1.  **反差**：开头和结尾的情绪最好是相反的（喜->悲，松->紧）。
2.  **节点**：明确每个情绪转折的具体事件。

## 示例输入
写一章“主角发现一直信任的师父其实是魔教卧底”的情绪细纲。
