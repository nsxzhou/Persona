# 342. 细纲：冲突升级阶梯 (Conflict Escalation)

## 提示词内容

```json
{
  "task": "escalate_conflict_scene",
  "participants": "MC vs Bully",
  "steps": [
    "Step 1: Verbal insult (Low intensity)",
    "Step 2: Physical shoving (Medium intensity)",
    "Step 3: Weapon drawn / Hostage taken (High intensity)",
    "Step 4: Lethal intent / Life or death (Climax)"
  ],
  "mc_reaction": "Trying to de-escalate -> Forced to fight -> Finishing blow",
  "audience_reaction": "Laughing -> Shocked -> Terrified"
}
```

## 使用场景
细纲/战斗/打脸。确保冲突有层次感，不是一上来就拼命。

## 最佳实践要点
1.  **层层递进**：每一步都比上一步更严重，让读者感到压力增加。
2.  **爆发点**：最后的高潮必须足够炸裂。

## 示例输入
设计一场“主角在餐厅被富二代挑衅”的冲突升级细纲。
