# 344. 细纲：信息释放节奏 (Information Drip)

## 提示词内容

```json
{
  "task": "plan_information_reveal",
  "secret": "The world is actually a simulation",
  "clues": [
    {"chapter": 10, "detail": "Birds flying in repetitive patterns"},
    {"chapter": 50, "detail": "A glitch in the sky texture"},
    {"chapter": 100, "detail": "MC finds an invisible wall at the edge of the world"}
  ],
  "reveal_moment": "MC meets the Admin",
  "misdirection": "Characters think it's a ghost or magic"
}
```

## 使用场景
细纲/悬疑/设定。控制世界观或谜题的揭晓速度。

## 最佳实践要点
1.  **碎片化**：不要一次性把设定全抛出来，要一点点喂给读者。
2.  **误导**：利用角色的认知局限来误导读者。

## 示例输入
设计“主角身世之谜”的信息释放节奏，每50章一个线索。
