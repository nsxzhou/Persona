# 345. 细纲：爽文节奏控制器 (Pacing Controller)

## 提示词内容

```json
{
  "task": "control_pacing_cycle",
  "cycle_type": "Suppression -> Explosion -> Reward -> Digest",
  "content": {
    "Suppression": "MC is ridiculed, resources stolen (3 chapters)",
    "Explosion": "MC breaks through, faces slapping everyone (1 chapter)",
    "Reward": "Looting the enemy, gaining fame (1 chapter)",
    "Digest": "Converting loot to power, planning next move (1 chapter)"
  },
  "intensity_curve": "Low -> High -> Medium -> Low"
}
```

## 使用场景
细纲/爽文。经典的网文节奏公式，保证爽点不断。

## 最佳实践要点
1.  **压抑要够**：没有压抑就没有爆发的爽感（欲扬先抑）。
2.  **消化**：爆发后必须有“消化/整理收获”的平缓期，调节读者疲劳。

## 示例输入
规划一个“宗门大比”的小高潮节奏，共6章。
