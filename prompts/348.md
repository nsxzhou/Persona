# 348. 细纲：对话潜台词设计 (Subtext in Dialogue)

## 提示词内容

```json
{
  "task": "write_dialogue_subtext",
  "context": "Two spies having dinner",
  "surface_topic": "The weather / The food",
  "subtext": "Do you have the intel? / Are we being watched?",
  "example": {
    "A": "This steak is a bit rare. (Situation is dangerous)",
    "B": "I prefer it well-done. (I want to abort)",
    "A": "Try the wine, it calms the nerves. (Stay calm, I have a plan)"
  }
}
```

## 使用场景
细纲/谍战/权谋/言情。话里有话，增加剧情张力。

## 最佳实践要点
1.  **双关语**：利用日常词汇指代特定的人或事。
2.  **情绪掩饰**：表面客气，内心杀气腾腾。

## 示例输入
设计一场“皇帝与权臣”的对话，表面聊下棋，实则聊兵权。
