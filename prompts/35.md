# 35. 爆款书名生成器 (男频专用)

## 提示词内容

```json
{
  "task": "generate_titles",
  "target_audience": "male",
  "genre": "{{genre}}",
  "core_hook": "{{core_hook}}",
  "count": 10,
  "formulas": [
    "角色+金手指+结果 (如: 狱医+透视+无敌)",
    "开局+状态+反转 (如: 开局被退婚+觉醒神脉)",
    "脑洞+修饰词+身份 (如: 疯狂+囤货+末世之王)",
    "情绪+动词+对象 (如: 震惊+全网+女帝)"
  ],
  "requirements": "书名需直白、有冲击力、包含核心卖点"
}
```

## 使用场景
番茄男频小说选题。生成符合男性读者喜好的直白爽文书名。

## 最佳实践要点
1.  **公式化生成**：利用经过市场验证的公式（如“开局+反转”），保证书名下限。
2.  **情绪词**：强调使用“震惊”、“疯狂”、“无敌”等高能词汇，吸引点击。

## 示例输入
将 `{{genre}}` 替换为“末世”，`{{core_hook}}` 替换为“只有我有安全屋”。
