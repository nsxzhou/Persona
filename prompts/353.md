# 353. 流程：场景卡转正文 (Scene to Text)

## 提示词内容

```json
{
  "task": "expand_scene_to_text",
  "scene_brief": "MC enters the tavern, orders a drink, and overhears a rumor.",
  "sensory_details": "Smell of stale beer / Dim light / Noise of drunkards",
  "dialogue_style": "Rough, slang-heavy",
  "internal_monologue": "MC is cautious, checking exits",
  "word_count_target": 500
}
```

## 使用场景
写作/扩写。将简略的场景描述扩写为有血有肉的正文。

## 最佳实践要点
1.  **五感**：加入视觉、听觉、嗅觉描写，增加沉浸感。
2.  **潜台词**：对话不要只为了传递信息，要展现人物性格。

## 示例输入
扩写场景：主角第一次踏入修仙界最大的城市“天机城”。
