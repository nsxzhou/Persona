# 354. 流程：对话逻辑补全 (Dialogue Filler)

## 提示词内容

```json
{
  "task": "fill_dialogue_gap",
  "context": "MC convinces the Merchant to lower the price",
  "start_line": "MC: 'This sword is rusted.'",
  "end_line": "Merchant: 'Fine, take it for 50 gold.'",
  "tone": "Witty / Aggressive / deceptive",
  "length": "5-8 exchanges"
}
```

## 使用场景
写作/润色。只知道开头结尾，让AI补全中间的对话逻辑。

## 最佳实践要点
1.  **拉扯**：对话要有来有回，像打乒乓球。
2.  **性格**：商人的精明和主角的狡猾要在对话中体现。

## 示例输入
补全对话：主角说服高冷女帝结盟。
