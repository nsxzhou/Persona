# 355. 流程：打斗动作细化 (Combat Choreography)

## 提示词内容

```json
{
  "task": "choreograph_fight_scene",
  "summary": "MC dodges a fireball and counters with a sword slash.",
  "style": "Wuxia / Anime / Realistic",
  "details": [
    "Footwork: Sidestep to the left",
    "Sensory: Heat of the flame passing by",
    "Impact: Sword cutting through robes",
    "Sound: Sizzling air"
  ]
}
```

## 使用场景
写作/战斗。将“A打了B一下”扩写为精彩的动作戏。

## 最佳实践要点
1.  **动词**：使用精准的动词（侧身、横扫、挑刺）。
2.  **节奏**：短句为主，营造紧张感。

## 示例输入
细化动作：主角用太极拳借力打力，震飞了壮汉。
