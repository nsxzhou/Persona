# 356. 流程：环境描写扩充 (Setting Expansion)

## 提示词内容

```json
{
  "task": "expand_setting_description",
  "brief": "A scary old house.",
  "mood": "Eerie / Abandoned / Gothic",
  "elements": [
    "Rotting wood structure",
    "Overgrown ivy like snakes",
    "Windows like hollow eyes",
    "Sound of creaking floorboards"
  ],
  "metaphor": "The house stood like a dying beast."
}
```

## 使用场景
写作/润色。将干瘪的背景描述变得生动形象。

## 最佳实践要点
1.  **比喻**：用比喻来强化氛围（像野兽、像骷髅）。
2.  **细节**：抓住一个细节（如剥落的墙皮）进行特写。

## 示例输入
扩写环境：一座悬浮在云端的仙家宫殿。
