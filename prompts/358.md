# 358. 流程：过渡段落生成 (Transition Generator)

## 提示词内容

```json
{
  "task": "generate_transition_paragraph",
  "scene_A": "MC finishes breakfast.",
  "scene_B": "MC arrives at the dungeon entrance.",
  "method": "Time skip / Travel montage / Introspection",
  "content": "Briefly describe the journey, the changing landscape, and MC's preparation.",
  "length": "100-200 words"
}
```

## 使用场景
写作/润色。连接两个场景，避免生硬的跳跃。

## 最佳实践要点
1.  **略写**：无意义的过程（如赶路）要略写，但不能没有。
2.  **铺垫**：利用过渡段落调整节奏，为下一个场景做心理铺垫。

## 示例输入
写一段从“宗门”到“秘境”的赶路过渡，体现路途遥远。
