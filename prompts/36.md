# 36. 爆款书名生成器 (女频专用)

## 提示词内容

```json
{
  "task": "generate_titles",
  "target_audience": "female",
  "genre": "{{genre}}",
  "core_hook": "{{core_hook}}",
  "count": 10,
  "formulas": [
    "萌宝/马甲+打脸+甜宠",
    "重生+复仇+虐渣",
    "读心/吃瓜+反差+群像",
    "特定身份+禁欲/病娇+互动"
  ],
  "requirements": "强调情绪价值、CP感或爽感"
}
```

## 使用场景
番茄女频小说选题。生成符合女性读者喜好的情感/爽文书名。

## 最佳实践要点
1.  **元素堆叠**：结合“萌宝”、“马甲”、“读心”等女频热词。
2.  **情感张力**：强调 CP 感（如“病娇”、“禁欲”）和情绪价值（“虐渣”、“甜宠”）。

## 示例输入
将 `{{genre}}` 替换为“现言”，`{{core_hook}}` 替换为“带球跑”。
