# 361. 流程：爽点强化 (Hook Sharpener)

## 提示词内容

```json
{
  "task": "sharpen_scene_hook",
  "original_scene": "MC wins the tournament.",
  "enhancement": [
    "Add an audience reaction shot (Shock)",
    "Add a rival's breakdown (Regret)",
    "Add a reward reveal (Greed)",
    "Add a twist (The prize is fake)"
  ],
  "goal": "Maximize the dopamine hit"
}
```

## 使用场景
润色/爽文。让原本平淡的胜利变得更爽。

## 最佳实践要点
1.  **侧面烘托**：爽感往往来自于他人的反应，而不是胜利本身。
2.  **情绪释放**：长期的压抑在这一刻得到彻底释放。

## 示例输入
强化“主角考试第一名”的爽点，增加班主任的反应。
