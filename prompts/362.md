# 362. 流程：悬念钩子植入 (Cliffhanger Injector)

## 提示词内容

```json
{
  "task": "inject_cliffhanger",
  "chapter_end": "MC goes to sleep.",
  "problem": "Boring ending",
  "fix_options": [
    "A mysterious knock on the door",
    "System alert: 'Emergency Mission'",
    "Realizing someone is under the bed",
    "Finding a bloody note on the pillow"
  ],
  "selection": "Option 2"
}
```

## 使用场景
写作/润色。拯救平淡的章节结尾，提高留存率。

## 最佳实践要点
1.  **突然性**：在最平静的时候打破平静。
2.  **危机感**：新的危机是最好的钩子。

## 示例输入
给这一章加个钩子：主角刚杀完BOSS，正准备休息。
