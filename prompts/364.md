# 364. 流程：配角加戏 (Side Character Flesh-out)

## 提示词内容

```json
{
  "task": "flesh_out_side_character",
  "character": "Shopkeeper NPC",
  "role": "Sells items",
  "quirk": "Always cleaning a glass / Has a robotic eye / Speaks in riddles",
  "backstory_hint": "A retired adventurer with a knee arrow",
  "interaction": "Giving MC a discount because he looks like his dead son"
}
```

## 使用场景
写作/人设。让工具人NPC变得生动，增加世界真实感。

## 最佳实践要点
1.  **记忆点**：给配角一个标志性的动作或口癖。
2.  **故事感**：只言片语暗示配角也有自己的过去。

## 示例输入
给新手村的“铁匠大叔”加戏，让他看起来不只是个NPC。
