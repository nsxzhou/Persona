# 366. 流程：情感爆发点设计 (Emotional Climax)

## 提示词内容

```json
{
  "task": "design_emotional_climax",
  "buildup": "MC suppresses his grief for 50 chapters.",
  "trigger": "Visiting his friend's grave / Finding a lost letter",
  "reaction": "Breaking down / Screaming / Silent tears",
  "imagery": "Rain mixing with tears / Fist pounding the ground",
  "dialogue": "A monologue to the deceased"
}
```

## 使用场景
细纲/写作。设计全书或卷末的泪点。

## 最佳实践要点
1.  **压抑**：前面的压抑越久，爆发越感人。
2.  **宣泄**：给主角一个彻底宣泄的场景，不要憋着。

## 示例输入
设计主角得知“一直冷漠的父亲其实深爱自己”时的情感爆发。
