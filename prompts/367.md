# 367. 流程：反转逻辑链构建 (Plot Twist Logic)

## 提示词内容

```json
{
  "task": "construct_plot_twist",
  "twist": "The helpful guide is the main villain.",
  "clues": [
    "He knew the map too well",
    "He disappeared during fights",
    "He smiled when MC got hurt"
  ],
  "reveal_scene": "MC finds the guide sitting on the villain's throne.",
  "logic_check": "Does this contradict previous scenes? (No, he was acting)"
}
```

## 使用场景
大纲/细纲。设计意料之外、情理之中的反转。

## 最佳实践要点
1.  **铺垫**：反转必须有迹可循，不能机械降神。
2.  **回顾**：揭晓时引导读者回顾之前的细节，产生恍然大悟感。

## 示例输入
设计一个“系统其实是寄生虫”的反转逻辑链。
