# 368. 流程：信息密度调整 (Info Density Check)

## 提示词内容

```json
{
  "task": "adjust_info_density",
  "paragraph": "[Paste Text]",
  "problem": "Too much exposition (Info dump)",
  "fix": "Break it down into dialogue / Spread it over 3 paragraphs / Show through action",
  "goal": "Keep the reader engaged without confusion"
}
```

## 使用场景
润色/写作。防止设定堆砌劝退读者。

## 最佳实践要点
1.  **稀释**：把干货设定掺杂在剧情和对话中喂给读者。
2.  **相关性**：只介绍当前剧情需要的设定，其他的以后再说。

## 示例输入
优化这一段关于“魔法世界历史”的长篇大论，改成对话形式。
