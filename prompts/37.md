# 37. 选题市场潜力评估 (深度版)

## 提示词内容

```json
{
  "task": "evaluate_idea",
  "idea_summary": "{{idea_summary}}",
  "evaluation_metrics": {
    "novelty": "创新度 (1-10)",
    "commercial_value": "商业潜力 (1-10)",
    "adaptation_potential": "短剧/漫改潜力 (1-10)",
    "hook_strength": "开篇吸引力 (1-10)"
  },
  "feedback_sections": [
    "亮点分析",
    "潜在毒点/劝退点",
    "优化建议 (如何更符合番茄读者口味)"
  ]
}
```

## 使用场景
创意验证阶段。对具体的脑洞进行多维度的商业评估。

## 最佳实践要点
1.  **多维打分**：引入“短剧潜力”评估，符合当前网文 IP 开发趋势。
2.  **毒点排查**：强制要求列出“潜在毒点”，帮助作者规避市场雷区。

## 示例输入
填入你的核心脑洞（300字以内）。
