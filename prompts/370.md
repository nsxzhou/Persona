# 370. 流程：完稿最终检查清单 (Final Polish Checklist)

## 提示词内容

```json
{
  "task": "final_polish_check",
  "checklist": [
    "Spelling and Grammar",
    "Pacing (Is any part too boring?)",
    "Dialogue tags (Avoid repetitive 'he said')",
    "Consistency (Names, Eye colors)",
    "Hook at the end"
  ],
  "action": "Read aloud to catch rhythm issues"
}
```

## 使用场景
润色/发布前。发布前的最后一道关卡。

## 最佳实践要点
1.  **朗读**：朗读能发现默读发现不了的语病和节奏问题。
2.  **排版**：检查段落是否过长，手机阅读体验是否友好。

## 示例输入
对这一章进行最后的发布前检查。
