# 372. 润色：吵架/撕逼逻辑链 (Argument Logic)

## 提示词内容

```json
{
  "task": "construct_argument_flow",
  "topic": "Who ate the last pudding / Who betrayed the team",
  "participants": "Character A (Aggressive) vs Character B (Defensive but guilty)",
  "escalation": [
    "Accusation",
    "Denial with bad excuse",
    "Bringing up old grudges",
    "Personal attacks",
    "Physical confrontation"
  ],
  "outcome": "Character B breaks down and confesses"
}
```

## 使用场景
写作/情感/冲突。让吵架不仅仅是泼妇骂街，而是有逻辑的博弈。

## 最佳实践要点
1.  **翻旧账**：真实的吵架往往会扯出以前的陈年旧事。
2.  **情绪失控**：从讲道理到不讲道理的过程。

## 示例输入
设计一场“情侣因为买房问题”引发的激烈争吵。
