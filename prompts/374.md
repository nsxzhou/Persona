# 374. 场景：梦核/阈限空间描写 (Liminal Space)

## 提示词内容

```json
{
  "task": "describe_liminal_space",
  "setting": "Empty school hallway at night / Endless hotel corridor / Indoor pool with no water",
  "atmosphere": "Nostalgic but unsettling / Artificial / Silence",
  "sensory_details": "Humming of fluorescent lights / Smell of chlorine / Muted colors",
  "anomaly": "A door that shouldn't be there / A shadow that doesn't move",
  "feeling": "You've been here before, but you don't know when."
}
```

## 使用场景
悬疑/规则怪谈/恐怖。营造那种“似曾相识但又不对劲”的诡异感。

## 最佳实践要点
1.  **无人**：必须是本来应该有人但现在没人的地方。
2.  **人工感**：强调灯光、瓷砖、地毯等人造物的质感。

## 示例输入
描写一个“空无一人的深夜游乐场”。
