# 376. 润色：战斗中的心理博弈 (Combat Psychology)

## 提示词内容

```json
{
  "task": "write_combat_psychology",
  "situation": "Swordsman vs Mage",
  "mc_thought": "He's charging a spell. Should I rush or dodge?",
  "enemy_action": "Feinting a fireball but casting a shield",
  "realization": "He knows I'm aggressive!",
  "counter_tactic": "MC throws his sword instead of rushing",
  "result": "Enemy shield breaks, MC wins"
}
```

## 使用场景
战斗/智斗。让战斗不仅仅是拼数值，而是拼脑子。

## 最佳实践要点
1.  **预判**：我预判了你的预判。
2.  **欺诈**：战斗就是通过假动作和信息差欺骗对手。

## 示例输入
描写一场“刺客对决”，双方都在试图找出对方的隐身位置。
