# 378. 类型：短视频/Reels 剧本 (Short Video Script)

## 提示词内容

```json
{
  "task": "write_short_video_script",
  "genre": "Comedy / Life Hack / Skit",
  "duration": "30-60 seconds",
  "hook": "First 3 seconds must grab attention (Visual or Audio)",
  "structure": "Setup -> Problem -> Twist -> Punchline",
  "visual_cues": "Fast cuts / Text on screen / Sound effects",
  "cta": "Call to action (Like & Subscribe)"
}
```

## 使用场景
新媒体/短剧/营销。适应短视频时代的快节奏脚本。

## 最佳实践要点
1.  **黄金3秒**：开头必须直接切入主题或制造悬念。
2.  **反转**：在短短几十秒内至少要有一个反转。

## 示例输入
写一个“程序员修电脑”的搞笑短视频脚本。
