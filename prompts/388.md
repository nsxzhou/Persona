# 388. 润色：甚至不用“说”字 (Dialogue Tags)

## 提示词内容

```json
{
  "task": "replace_said_tags",
  "dialogue": "'I hate you,' he said.",
  "alternatives": [
    "Action tag: 'I hate you.' He slammed the door.",
    "Descriptive tag: 'I hate you.' His voice cracked.",
    "No tag: 'I hate you.' (Implied by context)"
  ],
  "goal": "Remove repetitive 'said' / 'asked' / 'shouted'"
}
```

## 使用场景
润色/写作。提高文笔的流畅度，避免单调。

## 最佳实践要点
1.  **动作代替**：用动作描写来代替“他说”，既指明了说话人，又增加了画面感。
2.  **省略**：在双人对话中，可以直接省略标签。

## 示例输入
优化一段对话，去掉所有的“他说”、“她问道”。
