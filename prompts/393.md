# 393. 流派：时间管理大师 (Time Management)

## 提示词内容

```json
{
  "task": "write_time_management_schedule",
  "character": "MC dating 5 girls at once / CEO running 3 companies",
  "schedule": "Hour-by-hour breakdown",
  "crisis": "Two dates overlap / Meeting moved up",
  "solution": "Using a clone / Teleportation / Excuse generator",
  "tone": "Comedic / Tense thriller"
}
```

## 使用场景
都市/后宫/搞笑。高强度的多线操作。

## 最佳实践要点
1.  **紧凑**：时间表必须严丝合缝，容错率极低。
2.  **翻车边缘**：一直在翻车的边缘疯狂试探。

## 示例输入
主角要在情人节这一天陪三个女朋友，制定时间表。
