# 4. 黄金书名生成器 (A/B 测试版)

## 提示词内容

```json
{
  "task": "generate_book_titles",
  "core_hook": "{{core_hook}}",
  "genre": "{{genre}}",
  "target_audience": "{{audience}}",
  "strategies": [
    {
      "type": "Direct Benefit",
      "template": "Role + Cheat + Outcome"
    },
    {
      "type": "Contrast/Shock",
      "template": "Humiliation + Awakening + Revenge"
    },
    {
      "type": "Question/Suspense",
      "template": "Situation + Weird Phenomenon + Question"
    },
    {
      "type": "Meme/Trend",
      "template": "Hot Stem + Profession"
    }
  ],
  "count_per_strategy": 3
}
```

## 使用场景
在确定核心梗后使用。生成多组不同风格的书名，用于测试点击率（CTR）。

## 最佳实践要点
1.  **少样本提示 (Few-Shot Prompting)**：通过 `strategies` 中的模板提供隐式示例，引导 AI 生成特定结构的书名。
2.  **多样性策略**：一次性生成多种类型（利益型、反差型、悬念型），便于 A/B 测试。
3.  **参数化输入**：使用 `{{core_hook}}` 等占位符，使提示词可复用。

## 示例输入
将 `{{core_hook}}` 替换为“能听到古董的心声”，`{{genre}}` 替换为“都市鉴宝”，`{{audience}}` 替换为“18-35岁男性”。
