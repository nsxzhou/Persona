# 400. 完结：创作心路历程复盘 (Writer's Retrospective)

## 提示词内容

```json
{
  "task": "write_creative_retrospective",
  "questions": [
    "What was the original spark?",
    "What changed during writing?",
    "Which character surprised you the most?",
    "What was the hardest scene to write?"
  ],
  "tone": "Reflective / Honest / Humble",
  "goal": "Connect with aspiring writers and loyal fans"
}
```

## 使用场景
完本感言/采访/复盘。作者对创作过程的深度总结。

## 最佳实践要点
1.  **真实**：分享创作中的痛苦和瓶颈，拉近与读者的距离。
2.  **彩蛋**：透露一些废弃的大纲或设定。

## 示例输入
写一篇复盘，讲述这本小说是如何从一个废弃的脑洞变成百万字长篇的。
