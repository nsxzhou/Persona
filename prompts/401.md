# 401. 流派：多子多福系统 (Many Children System)

## 提示词内容

```json
{
  "task": "design_many_children_system",
  "mechanic": "Reward per child born",

    "Cultivation base (+10 years)",
    "Lifespan (+50 years)",
    "Family Buff (Spirit Qi density +10%)",
    "Special Item (Legendary Artifact)"
  ],
  "feedback_loop": "More children -> Stronger MC -> Better partners -> Better children",
  "restriction": "Must be MC's biological offspring"
}
```

## 使用场景
多子多福/家族修仙。该流派的核心爽点在于“生孩子=变强”。

## 最佳实践要点
1.  **滚雪球**：强调指数级的增长，前期艰难，后期躺赢。
2.  **数值反馈**：每生一个孩子，都要有明确的数值提升（寿命、修为）。

## 示例输入
设计一个“纳妾就变强，生子更加倍”的系统面板。
