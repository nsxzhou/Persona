# 402. 流派：家族养成/种田 (Clan Management)

## 提示词内容

```json
{
  "task": "write_clan_management",
  "stage": "Early stage (Small family) -> Late stage (Immortal Clan)",
  "resources": "Spirit Stones / Mines / Herb Gardens",
  "policy": [
    "Encourage birth (Rewards for having kids)",
    "Education system (Academy)",
    "Marriage alliances"
  ],
  "crisis": "Rival clan invasion / Beast tide",
  "mc_role": "Old Ancestor (protecting from shadows) / Clan Leader"
}
```

## 使用场景
家族文/种田文。重点在于家族势力的整体提升。

## 最佳实践要点
1.  **群像**：除了主角，要塑造几个有出息（或败家）的后代。
2.  **护短**：老祖宗（主角）在关键时刻出手护犊子的爽点。

## 示例输入
主角制定“家族百年发展计划”，首要任务是人口翻倍。
