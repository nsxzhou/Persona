# 409. 场景：直播荒野求生 (Wilderness Survival)

## 提示词内容

```json
{
  "task": "write_survival_stream",
  "environment": "Rainforest / Desert / Island / Arctic",
  "challenge": "Beast Attack / Shelter Building / Finding Water",
  "skill": "Primitive Tech / Combat / Knowledge",
  "interaction": "Audience Shock / Tipping / Memes (Danmu)",
  "climax": "Protagonist kills beast / Eats gourmet food",
  "outcome": "Subscribers explode / System reward"
}
```

## 使用场景
荒野直播/系统文。在极端环境下展示从容不迫的生存技能。

## 最佳实践要点
1.  **反差**：环境的恶劣与主角的享受（吃播、撸猫）形成对比。
2.  **弹幕互动**：通过观众的吐槽和震惊来侧面烘托主角的牛逼。

## 示例输入
主角在亚马逊雨林遭遇美洲豹，单杀后生火烤肉，观众弹幕疯狂刷“全网最硬核吃播”。
