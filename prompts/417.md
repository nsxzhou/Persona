# 417. 流派：无敌流/满级开局 (Invincible Start)

## 提示词内容

```json
{
  "task": "write_invincible_scene",
  "identity": "Hidden Master / Max Level Account / Retired Boss",
  "crisis": "Sect Destroyed / Monster Attack / Genius Defeated",
  "enemy": "Peerless Genius / Ancestor / Ancient Beast",
  "action": "One Finger Kill / Word of Law / Sigh",
  "reaction": "Enemy despair / Bystanders dumbfounded",
  "outcome": "Crisis solved instantly / Low profile maintained (failed)"
}
```

## 使用场景
玄幻/仙侠。跳过修炼过程，直接享受“满级大号屠新手村”的快感。

## 最佳实践要点
1.  **云淡风轻**：主角出手越随意（如拍苍蝇），装逼效果越好。
2.  **侧面烘托**：先极力描写敌人的强大和绝望，再由主角一招解决。

## 示例输入
魔教老祖攻打宗门，无人能敌，扫地杂役（主角）叹了口气，一扫帚将其拍进地里扣都扣不出来。
