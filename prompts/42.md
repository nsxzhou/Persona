# 42. 百万字长篇节奏规划 (阶段式)

## 提示词内容

```json
{
  "task": "long_term_pacing",
  "total_words": "100万字",
  "structure": {
    "phase_1": "新手村/底层逆袭 (0-20万字)",
    "phase_2": "市级/中层扬名 (20-50万字)",
    "phase_3": "跨界/高层争霸 (50-80万字)",
    "phase_4": "世界/位面巅峰 (80-100万字)"
  },
  "requirements": "定义每个阶段的核心目标、主要反派势力、金手指升级状态"
}
```

## 使用场景
长篇架构。通过地图换挡和能力升级来维持长篇热度。

## 最佳实践要点
1.  **地图换挡**：明确从“新手村”到“位面巅峰”的地图切换逻辑。
2.  **目标迭代**：每个阶段必须有新的核心目标和反派，避免剧情重复。

## 示例输入
无。
