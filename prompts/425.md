# 425. 同人：体育竞技 (Sports Fanfic)

## 提示词内容

```json
{
  "task": "write_sports_match",
  "sport": "NBA / Football",
  "opponent": "Jordan / Kobe / Messi / Ronaldo",
  "cheat": "Skill Card / System Training / Future Body",
  "moment": "Buzzer Beater / Posterize / Dribble Past 5",
  "reaction": "Opponent stunned / Crowd goes wild",
  "outcome": "Victory / Records broken / Dynasty begins"
}
```

## 使用场景
同人/竞技。穿越回过去，与传奇巨星同台竞技并战胜他们。

## 最佳实践要点
1.  **数据流**：用具体的数据（得分、篮板）来体现统治力。
2.  **名场面复刻/破坏**：在经典比赛中改变结果，或者创造新的名场面。

## 示例输入
主角穿越回96年总决赛，在乔丹头上完成“世纪之扣”，并投中绝杀球，抢走FMVP。
