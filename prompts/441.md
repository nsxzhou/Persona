# 441. 同人创作：选题与切入点 (Fanfic Ideation)

## 提示词内容

```json
{
  "task": "fanfic_ideation",
  "fandom": "Naruto / Harry Potter / One Piece / Genshin",
  "trope": "Canon Divergence / Modern AU / Crossover / Fix-it",
  "protagonist_type": "Reincarnator (Knows Plot) / Native OC / Canon Character (Peggy Sue)",
  "golden_finger": "System / Bloodline / Knowledge / None",
  "core_conflict": "Survival / Changing Fate / Harem / World Domination",
  "hook": "What if [Character] didn't die? / What if [Event] happened differently?",
  "output_format": "Title + Logline + Core Selling Point"
}
```

## 使用场景
同人创作第一步。确定写哪个IP，什么类型，以及核心爽点是什么。

## 最佳实践要点
1.  **脑洞差异化**：避免千篇一律的开局，寻找“如果...会怎样”的独特切入点。
2.  **受众匹配**：明确是写给考据党看（严谨风）还是写给爽文党看（无敌流）。

## 示例输入
我想写一本火影同人，主角是卡卡西的弟弟，切入点是“如果旗木朔茂没有自杀”，核心爽点是改变白牙之死的悲剧，建立强大的旗木家族。
