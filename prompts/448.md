```json
  "chemistry": "Banter / Fluff / Angst / Steam",
  "change": "How the relationship affects the main plot"
}
```

## 使用场景
同人创作第八步。处理主角的感情线，或者撮合/拆散原著CP。

## 最佳实践要点
1.  **尊重原著情感**：如果拆官配，要有合理的铺垫和理由，不能硬拆。
2.  **角色魅力**：让原著角色爱上主角，必须是因为主角展现了独特的魅力（亚撒西/强大/理解）。

## 示例输入
主角（男）与灰原哀。在一次爆炸案中，主角不顾性命救出小哀，小哀在包扎伤口时红着脸骂主角是“笨蛋”。
