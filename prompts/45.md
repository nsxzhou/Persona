# 45. 黄金简介撰写 (三版对比)

## 提示词内容

```json
{
  "task": "write_blurbs",
  "book_info": "{{book_info}}",
  "versions": [
    {
      "style": "悬念型",
      "focus": "强调开局的危机和谜团"
    },
    {
      "style": "爽文型",
      "focus": "强调金手指的强大和打脸快感"
    },
    {
      "style": "人设型",
      "focus": "强调主角的特殊身份或反差萌"
    }
  ],
  "structure": "三句话吸睛 + 正文片段展示 + 引导语"
}
```

## 使用场景
简介创作。生成三个不同版本的简介供选择。

## 最佳实践要点
1.  **风格对比**：同时提供悬念、爽文、人设三种风格，满足不同侧重点的需求。
2.  **引导语**：包含“引导语”，增加读者的点击欲望。

## 示例输入
填入书籍基本信息。
