# 451. 润色：角色语气模仿 (Character Voice Mimicry)

## 提示词内容

```json
{
  "task": "mimic_character_voice",
  "character": "Naruto / Zhongli / Tony Stark / Frieza",
  "trait": "Verbal Tic (Dattebayo) / Archaic / Sarcastic / Polite Evil",
  "context": "Ordering food / Declaring war / Comforting a friend",
  "dialogue_content": "Draft content (e.g., 'Don't give up')",
  "style_guide": "Sentence length, vocabulary choice, tone",
  "output": "3 Variations of the dialogue in character"
}
```

## 使用场景
同人/对话润色。确保角色说话“有那味儿”，避免OOC（Character Acting）。

## 最佳实践要点
1.  **口癖与关键词**：抓住角色的标志性口癖（如“杂修”、“本大爷”）。
2.  **句式结构**：比如钟离喜欢用长句和成语，而路飞喜欢用短句和感叹号。

## 示例输入
请用“钟离”（原神）的语气，评价现代的汉堡包。要求体现出他的博学、沉稳以及对价格的脱节感。
