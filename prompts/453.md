# 453. 润色：原著场景还原 (Iconic Location Immersion)

## 提示词内容

```json
{
  "task": "describe_canon_location",
  "location": "Hogwarts Great Hall / Konoha Hokage Rock / Going Merry Deck",
  "time": "Sunrise / Midnight / Festival",
  "sensory_details": "Smell (Food/Sea), Sound (Chatter/Waves), Light",
  "atmosphere": "Cozy / Majestic / Nostalgic",
  "canon_elements": "Floating candles / Graffiti on faces / Tangerine trees",
  "mc_perspective": "First time seeing it vs Living there"
}
```

## 使用场景
同人/环境。通过细节描写唤起读者的DNA，增强代入感。

## 最佳实践要点
1.  **标志性细节**：必须提到原著中该场景的独特物品（如霍格沃茨的漂浮蜡烛）。
2.  **氛围营造**：不仅仅是物理描写，更是描写该场景带给人的“感觉”（如家的感觉）。

## 示例输入
描写主角第一次走进“霍格沃茨礼堂”的场景。重点描写施了魔法的天花板（映照着夜空）和四张长桌上堆满的美食香气。
