# 454. 润色：情感微表情与心理 (Romantic Micro-Expressions)

## 提示词内容

```json
{
  "task": "write_micro_expressions",
  "character": "Tsundere / Kuudere / Genki",
  "emotion": "Hidden Love / Jealousy / Shy / Heartbreak",
  "trigger": "Hand touching / Compliment / Seeing rival",
  "action": "Looking away / Clenching fist / Red ears / Stuttering",
  "internal_monologue": "Denial vs Truth",
  "subtext": "What is not said aloud"
}
```

## 使用场景
同人/CP文。通过细腻的肢体语言和微表情来发糖或发刀。

## 最佳实践要点
1.  **非语言沟通**：眼神的躲闪、耳根的红晕往往比直接表白更动人。
2.  **反差**：傲娇角色的嘴硬与身体诚实（如嘴上骂笨蛋，手却抓着衣角不放）。

## 示例输入
描写御坂美琴（某科学的超电磁炮）在被主角夸奖“今天的发卡很可爱”时的反应。要求体现傲娇属性，脸红但嘴硬。
