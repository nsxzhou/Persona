# 458. 润色：原著角色震惊反应 (Canon Reaction POV)

## 提示词内容

```json
{
  "task": "write_shock_reaction",
  "observer": "Canon Character (Expert / Rival / Mentor)",
  "trigger": "MC reveals hidden power / Solves impossible problem",
  "internal_thought": "Disbelief -> Analysis -> Fear/Awe",
  "physical_reaction": "Widened eyes / Dropped weapon / Trembling",
  "dialogue": "'Impossible...' / 'That technique is...!'",
  "comparison": "Comparing MC to a legend (e.g., 'He's just like the 4th Hokage')"
}
```

## 使用场景
同人/装逼打脸。通过原著土著（特别是强者）的视角来侧面吹捧主角。

## 最佳实践要点
1.  **专业解说**：让懂行的角色（如卡卡西、解说员）来分析主角的操作有多难。
2.  **脑补**：让原著角色自我攻略，脑补主角的背景或深意。

## 示例输入
以卡卡西的视角，描写他看到主角施展“飞雷神之术”时的心理活动。从震惊到回忆起老师（波风水门），最后转化为对主角的认可。
