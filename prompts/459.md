# 459. 润色：日常温馨与节日 (Slice of Life)

## 提示词内容

```json
{
  "task": "write_slice_of_life",

  "atmosphere": "Warm / Chaos / Funny / Nostalgic",
  "highlight": "A moment of peace before the storm",
  "interaction": "Banter / Pranks / Heart-to-heart"
}
```

## 使用场景
同人/日常。调节节奏，描写角色之间的羁绊和美好时光。

## 最佳实践要点
1.  **群像互动**：让每个角色都展现出其性格特点（如路飞偷吃、娜美暴怒、索隆喝酒）。
2.  **暴风雨前的宁静**：温馨的日常往往是为了衬托后续战斗的残酷，或者作为战斗后的治愈。

## 示例输入
描写草帽海贼团在桑尼号上的午后日常。路飞和乌索普在钓鱼，山治在做饮料，索隆在睡觉，罗宾在看报纸，一片祥和。
