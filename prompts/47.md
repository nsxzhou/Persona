# 47. 内容合规性自查

## 提示词内容

```json
{
  "task": "compliance_check",
  "content_summary": "{{content}}",
  "check_list": [
    "涉政/涉黑/暴恐",
    "低俗色情/擦边球",
    "封建迷信 (过度宣扬)",
    "三观不正 (如美化犯罪)",
    "侵权/同人越界"
  ],
  "output": "风险等级评估 + 修改建议"
}
```

## 使用场景
发布前检查。规避平台审核风险。

## 最佳实践要点
1.  **风险清单**：明确列出常见的审核红线（涉政、色情等）。
2.  **修改建议**：不仅指出风险，还提供修改建议，确保内容既安全又保留看点。

## 示例输入
填入大纲或章节摘要。
