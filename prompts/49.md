# 49. 变异生物图鉴生成

## 提示词内容

```json
{
  "task": "generate_monster_bestiary",
  "environment": "{{environment}}",
  "threat_level": "Low / Medium / Nightmare",
  "count": 3,
  "output_structure": {
    "name": "Creative Name (e.g., Iron-skinned Boar)",
    "base_creature": "Original animal/plant",
    "mutation_feature": "Visual horror/cool factor",
    "attack_pattern": "How it hunts",
    "weakness": "Critical spot for MC to exploit",
    "loot": "Valuable material dropped"
  }
}
```

## 使用场景
科幻/末世/玄幻。设计用于战斗或提供资源的怪物。

## 最佳实践要点
1.  **生态合理性**：怪物的 `mutation_feature` 应与 `environment` 挂钩（如沙漠生物耐热）。
2.  **战斗交互**：设计明确的 `attack_pattern` 和 `weakness`，让战斗描写更有策略性。

## 示例输入
将 `{{environment}}` 替换为“辐射废土”。
