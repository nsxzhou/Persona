# 55. 网文转短剧剧本 (场景化)

## 提示词内容

```json
{
  "task": "adapt_novel_to_script",
  "source_text": "{{novel_excerpt}}",
  "format": "Short Video Script (Vertical Screen)",
  "constraints": [
    "Fast pacing (Hook in first 3 seconds)",
    "Visual storytelling (Show, don't tell)",
    "Exaggerated emotions"
  ],
  "output_structure": {
    "Scene 1": {
      "Location": "...",
      "Time": "...",
      "Action": "...",
      "Dialogue": "...",
      "Camera_Angle": "Close-up on MC's sneer"
    }
  }
}
```

## 使用场景
IP改编/短剧剧本创作。将小说文本转换为拍摄脚本。

## 最佳实践要点
1.  **视觉化**：强调“Show, don't tell”，将心理活动转化为动作或表情。
2.  **黄金前三秒**：短剧特性，开头必须有强视觉冲击或冲突。

## 示例输入
填入一段爽文高潮片段。
