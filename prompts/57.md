# 57. 克苏鲁风格描写 (San值狂掉)

## 提示词内容

```json
{
  "task": "write_lovecraftian_description",
  "subject": "{{subject}}",
  "keywords": ["Non-euclidean", "Tentacles", "Madness", "Viscous", "Unspeakable"],
  "sensory_focus": "Smell of decay, Sound of wet slithering, Visual distortion",
  "psychological_effect": "MC feels insignificant, nauseous, questioning reality",
  "instruction": "Describe the subject using vague, disturbing adjectives. Avoid describing it directly."
}
```

## 使用场景
克苏鲁/诡秘流。描写不可名状的恐怖事物。

## 最佳实践要点
1.  **侧面描写**：强调“Avoid describing it directly”，用形容词和感受来堆砌恐怖感。
2.  **心理投射**：重点描写主角的生理不适和精神崩溃，而非怪物本身。

## 示例输入
将 `{{subject}}` 替换为“古神的雕像”。
