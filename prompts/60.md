# 60. 卡文紧急救援指南

## 提示词内容

```json
{
  "task": "overcome_writer_block",
  "current_situation": "{{summary_of_stuck_point}}",
  "strategies": [
    "Introduce a new character (The 'Man with a Gun' rule)",
    "Kill off a character",
    "Reveal a secret",
    "Force MC to make a difficult choice",
    "Change the setting/location"
  ],
  "instruction": "Based on the current situation, provide 3 specific plot twists to break the deadlock."
}
```

## 使用场景
写作中途卡文。提供破局的灵感。

## 最佳实践要点
1.  **雷蒙德·钱德勒定律**：“当不知道该写什么时，让一个人拿着枪进来。”（引入新冲突）。
2.  **具体方案**：根据当前剧情生成 3 个具体的转折点，而非抽象建议。

## 示例输入
填入卡文的剧情点（如“主角打败了反派，接下来没事干了”）。
