# 61. 暧昧期推拉感设计

## 提示词内容

```json
{
  "task": "write_romantic_tension",
  "stage": "Flirting / Ambiguous",
  "characters": ["A (Cold/Reserved)", "B (Playful/Proactive)"],
  "scenario": "Sharing an umbrella / Accidental touch / Late night office",
  "techniques": [
    "Physical proximity (Close but not touching)",
    "Double entendre dialogue",
    "Lingering eye contact",
    "Sensory focus (Smell of perfume, warmth)"
  ],
  "outcome": "Heart racing, but no confession yet"
}
```

## 使用场景
言情/CP文。描写两人关系未挑明时的极限拉扯。

## 最佳实践要点
1.  **非语言交流**：强调眼神、呼吸、体温等感官细节，而非直白的语言。
2.  **克制**：关键在于“No confession yet”，保持张力不崩。

## 示例输入
无。
