# 62. 追妻/夫火葬场节奏表

## 提示词内容

```json
{
  "task": "outline_redemption_arc",
  "trope": "Crematorium (Groveling for forgiveness)",
  "offense": "Misunderstanding / Neglect / Betrayal (Redeemable)",
  "stages": {
    "Stage 1": "Realization (MC leaves, Regret sets in)",
    "Stage 2": "Suffering (Drunk dialing, Standing in rain, Jealousy)",
    "Stage 3": "Sacrifice (Risking life/career for MC)",
    "Stage 4": "Acceptance (MC forgives but maintains boundaries)"
  },
  "satisfaction_factor": "Ensure the offender suffers enough before forgiveness"
}
```

## 使用场景
虐恋/破镜重圆文。规划一方悔过自新的过程。

## 最佳实践要点
1.  **虐度平衡**：前期的错误必须是可原谅的（非原则性错误），否则无法洗白。
2.  **受虐展示**：具体描写悔过方的惨状，满足读者的报复心理。

## 示例输入
无。
