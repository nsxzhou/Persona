# 65. 电竞文比赛解说流

## 提示词内容

```json
{
  "task": "write_esports_commentary",
  "game_type": "MOBA (LoL/Dota) / FPS (CS/Valorant)",
  "situation": "Clutch Play / Teamfight / Base Race",
  "commentators": ["A (Hype man)", "B (Analyst)"],
  "content": [
    "Play-by-play description (Fast)",
    "Skill interaction analysis",
    "Crowd reaction",
    "MC's god-tier micro"
  ],
  "style": "Passionate, Professional, Slang-heavy"
}
```

## 使用场景
电竞文。通过解说视角描写激烈的比赛过程。

## 最佳实践要点
1.  **专业术语**：使用正确的游戏术语（Gank, Kiting, Ult），增加真实感。
2.  **节奏带动**：解说员的激情呐喊能有效带动读者情绪。

## 示例输入
将 `{{game_type}}` 替换为“王者荣耀”。
