# 68. 综艺节目流程设计

## 提示词内容

```json
{
  "task": "design_variety_show",
  "type": "Dating / Survival / Talent",
  "name": "{{show_name}}",
  "cast": ["MC (The Wildcard)", "Idol (Traffic)", "Veteran (Respect)", "Comedian (Relief)"],
  "segments": [
    "Intro/Entrance",
    "Game 1: Icebreaker (Awkwardness)",
    "Game 2: Physical/Mental Challenge (MC shines)",
    "Lunch/Break (Character bonding)",
    "Final Showdown"
  ],
  "highlight": "The moment MC goes viral"
}
```

## 使用场景
娱乐圈文。设计综艺情节，展示主角魅力。

## 最佳实践要点
1.  **人设碰撞**：利用不同嘉宾的性格反差制造笑点或冲突。
2.  **高光时刻**：明确设计主角爆红的那个“梗”或“名场面”。

## 示例输入
将 `{{show_name}}` 替换为“荒野求生30天”。
