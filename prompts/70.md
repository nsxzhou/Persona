# 70. 幕后黑手流剧本编排

## 提示词内容

```json
{
  "task": "script_mastermind_plot",
  "goal": "Revive magic / Save the world / Amuse self",
  "fake_history": "Ancient war between Gods and Demons",
  "npc_roles": [
    "The 'Chosen One' (MC's puppet)",
    "The 'Dark Lord' (Also MC's puppet)",
    "The Historian (Discovers the 'truth')"
  ],
  "stage_1": "Create anomalies (rumors)",
  "stage_2": "Summon players/NPCs to explore",
  "stage_3": "Stage a grand event confirming the fake history"
}
```

## 使用场景
幕后流/马甲流。主角编造神话、忽悠全世界。

## 最佳实践要点
1.  **双重身份**：区分主角本体（幕后）和马甲（台前）。
2.  **弄假成真**：通过众人的信以为真，让谎言变成现实（如信仰成神）。

## 示例输入
无。
