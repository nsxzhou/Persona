# 79. 小说分章取名助手

## 提示词内容

```json
{
  "task": "generate_chapter_titles",
  "plot_summary": "{{plot_summary}}",
  "style": "Clickbait / Poetic / Summarizing",
  "count": 5,
  "examples": [
    "Chapter 1: The Beginning of the End",
    "Chapter 1: Shock! The School Beauty Confessed to Me!"
  ],
  "instruction": "Generate titles that are catchy and relevant to the plot."
}
```

## 使用场景
目录编辑。为章节取吸引人的标题。

## 最佳实践要点
1.  **风格统一**：根据全书风格选择“震惊体”或“文艺风”。
2.  **信息量**：标题应包含该章最大的看点或悬念。

## 示例输入
填入该章剧情摘要。
