# 82. 炼丹/炼器步骤生成器

## 提示词内容

```json
{
  "task": "generate_alchemy_process",
  "item_name": "Nine-Turn Resurrection Pill",
  "ingredients": [
    {"name": "Dragon Blood Grass", "role": "Main"},
    {"name": "Soul Flower", "role": "Catalyst"}
  ],
  "steps": [
    "Purification (Removing impurities)",
    "Fusion (Controlling fire)",
    "Condensation (Forming the pill)",
    "Tribulation (Lightning strikes the pill)"
  ],
  "failure_risk": "Explosion if temperature varies by 1 degree"
}
```

## 使用场景
修仙/玄幻文。描写炼丹或炼器的详细过程。

## 最佳实践要点
1.  **专业感**：通过复杂的步骤（提纯、融合、成丹）体现主角的技术含量。
2.  **异象**：丹成之时必有天地异象（如丹劫），烘托宝物的珍贵。

## 示例输入
将 `item_name` 替换为“筑基丹”。
