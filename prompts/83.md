# 83. 系统商城定价策略

## 提示词内容

```json
{
  "task": "design_system_shop",
  "currency": "Fear Points / Reputation / Spirit Stones",
  "items": [
    {"name": "Basic HP Potion", "price": "10", "description": "Heals minor wounds"},
    {"name": "Revival Token", "price": "1,000,000", "description": "One extra life"},
    {"name": "Mystery Box", "price": "100", "description": "Random item"}
  ],
  "inflation_control": "Items get more expensive as MC levels up?"
}
```

## 使用场景
系统流。设计合理的商城经济体系。

## 最佳实践要点
1.  **价格锚点**：设置一个极其昂贵的终极道具（如复活币），作为主角的长期动力。
2.  **消费陷阱**：设计“盲盒”或“限时折扣”，诱导主角消费，推动剧情。

## 示例输入
无。
