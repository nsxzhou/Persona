# 84. 美食文烹饪描写 (中华小当家风)

## 提示词内容

```json
{
  "task": "write_cooking_scene",
  "dish": "Golden Fried Rice / Mapo Tofu",
  "effect": "Glowing / Aphrodisiac / Memory Recall",
  "steps": [
    "Knife work (Fast as lightning)",
    "Wok Hei (Controlling the fire)",
    "Plating (Artistic)"
  ],
  "reaction": "Judges crying, clothes ripping (metaphorically), seeing illusions"
}
```

## 使用场景
美食文。描写夸张且诱人的烹饪过程和试吃反应。

## 最佳实践要点
1.  **通感**：用视觉（发光）、听觉（爆炒声）来写味觉。
2.  **夸张反应**：试吃者的反应必须极其夸张（如看到幻觉、流泪），体现美食的魔力。

## 示例输入
将 `dish` 替换为“开水白菜”。
