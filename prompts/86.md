# 86. 灵气复苏开局众生相

## 提示词内容

```json
{
  "task": "describe_mana_recovery_start",
  "anomalies": [
    "Pets speaking human language",
    "Plants growing wildly overnight",
    "Ancient mountains appearing from thin air"
  ],
  "social_chaos": "Panic buying, cults forming, military deployment",
  "mc_action": "Quietly hoarding resources while others panic"
}
```

## 使用场景
灵气复苏流。描写世界巨变初期的混乱与机遇。

## 最佳实践要点
1.  **宏大叙事**：通过多视角的社会乱象（动植物变异、社会停摆）烘托末世感。
2.  **先知优势**：强调主角的冷静与大众的恐慌形成对比。

无。
