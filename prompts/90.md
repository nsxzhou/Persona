# 90. 魔法学院分院仪式

## 提示词内容

```json
{
  "task": "design_sorting_ceremony",
  "method": "Magic Hat / Crystal Ball / Spirit Animal choice",
  "houses": [
    {"name": "House of Valor", "trait": "Brave"},
    {"name": "House of Wisdom", "trait": "Smart"},
    {"name": "House of Ambition", "trait": "Cunning"},
    {"name": "House of Shadow", "trait": "Mysterious (MC's house)"}
  ],
  "mc_result": "The artifact is confused / Breaks / Assigns a forbidden house"
}
```

## 使用场景
魔法/西幻学院流。设计标志性的分院情节。

## 最佳实践要点
1.  **主角特殊化**：主角的分院过程必须出现异常（如神器炸裂、双学院特质），凸显与众不同。
2.  **学院风格**：通过学院特质确立后续的阵营对立关系。

## 示例输入
无。
