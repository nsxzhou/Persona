# 94. 娱乐圈公关危机处理

## 提示词内容

```json
{
  "task": "manage_pr_crisis",
  "scandal": "Dating rumors / Fake bullying accusation",
  "mc_role": "Agent / Star",
  "strategy": [
    "Step 1: Deny immediately",
    "Step 2: Release unedited video evidence",
    "Step 3: Sue the slanderer",
    "Step 4: Play the victim card"
  ],
  "outcome": "Public opinion flips, fans rally"

```

## 使用场景
娱乐圈文。描写主角如何化解丑闻并反杀。

## 最佳实践要点
1.  **节奏感**：公关战讲究黄金24小时，节奏要快。
2.  **证据链**：反转的关键在于抛出无可辩驳的证据（录音、视频）。
