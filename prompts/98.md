# 98. 异能/魔法副作用设计

## 提示词内容

```json
{
  "task": "design_power_drawback",
  "power": "Invisibility / Super Strength / Time Travel",
  "cost": "Sanity / Aging / Calories / Pain",
  "trigger": "Usage over 5 minutes / Every activation",
  "mitigation": "Wearing special gear / Drinking potions"
}
```

## 使用场景
异能/超英/魔法文。平衡战斗力，增加剧情张力。

## 最佳实践要点
1.  **等价交换**：越强的力量，副作用（代价）越惨痛，避免战力崩坏。
2.  **克服过程**：主角寻找减轻副作用的方法（如寻找神器），本身就是一条主线。

## 示例输入
无。
